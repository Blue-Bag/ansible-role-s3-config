S3-config
=========

Configures a host to use an eponymous S3 Bucket for off site storage
http://docs.ansible.com/s3_module.html

Requirements
------------
Require an S3 account

Role Variables
--------------


Dependencies

Example Playbook
----------------

Including an example of how to use:

      roles:
